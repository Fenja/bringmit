Dies ist bringmit, eine einfache Möglichkeit, um Buffets für Veranstaltungen zu organisieren.
Um eine neue Veranstaltung zu erstellen, einfach auf den Button klicken.
Es können Titel, Datum und ein Beschreibungstext eingegeben werden, danach kann ein Link an alle Teilnehmer der Veranstaltung verschickt werden. TODO Bild einfügen
Falls die Informationen der Veranstaltung nachträglich noch geändert werden müssen, einfach die ID eingeben. Die Änderungen können dann gespeichert werden und sind sofort sichtbar.
Veranstaltungen werden eine Woche nach Ablauf automatisch gelöscht.
